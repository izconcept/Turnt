# Turnt

### IOT Automatic Drink Recipe Mixer

```
~ python3 -m venv .env
```

```
source .env/bin/activate
```

```
~ sudo pip install -r requirements.txt
```

```
deactivate
```

```
chmod a+x run.py
```

```
./run.py
```
